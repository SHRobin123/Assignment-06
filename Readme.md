📌Project Name:- DigiTools – Supercharge Your Digital Workflow
DigiTools is a modern and beautifully designed digital product marketplace UI where users can browse premium tools, add items to cart, and manage their checkout experience smoothly.
The project focuses on clean UI, smooth interactions, and a professional product listing system.

📝 Project Description
DigiTools is a responsive web interface built to showcase digital tools with a premium experience.
It includes product categories, detailed product cards, a dynamic cart section, and a fully animated toast notification system.
This design can be extended into a real SaaS marketplace or digital shop.

🛠️ Technologies Used
HTML5 – Page structure

CSS3 – Complete custom styling (no frameworks)

JavaScript (ES6+) – Logic for cart system, interactions, toast notifications

React.js + ReactDOM – Used through CDN to render UI components

Manrope Google Font – Modern typography

Babel Standalone – For JSX support in the browser

⭐ Key Features
✅ 1. Dynamic Shopping Cart
Users can add or remove digital products, and total price updates instantly.

✅ 2. Beautiful Toast Notification System
Custom-designed success, error, and warning alerts with animation effects.

✅ 3. Modern & Responsive Product Showcase
Clean UI with categories, product tags (New, Popular, Bestseller), and smooth hover animations.
